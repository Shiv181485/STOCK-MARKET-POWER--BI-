# 📌 Project Overview

This Power BI dashboard provides an interactive analysis of the Indian Stock Market using historical stock price and trading volume data.

The report helps investors and analysts monitor:

- 📊 Stock Performance
- 📈 Price Trends
- 💹 Trading Volume
- 📉 Volatility Analysis
- 🔥 Bullish vs Bearish Trends
- 🏢 Sector-wise Performance
- 📅 Monthly & Daily Insights

✅ July experienced the highest market volume.

✅ OFSS has one of the highest average closing prices.
