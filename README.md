# 📈 Stock Market Analytics Dashboard

## Overview
Interactive Power BI dashboard built using Indian Stock Market data to analyze stock performance, volatility, trading volume, price trends, and sector-wise insights.

## Features
- Interactive Symbol & Date Filters
- Stock Price Trend Analysis
- Trading Volume Analysis
- Volatility Analysis
- Bullish vs Bearish Market Trends
- Sector-wise Performance
- Monthly Price Change Analysis
- Price Range & Closing Price Analysis

## Tools
- Power BI Desktop
- DAX
- Power Query
- Excel

## Key Insights
- Agriculture sector recorded the highest average volatility.
- Railways sector generated the highest trading volume.
- July recorded the highest overall market volume.
- Bullish stocks outnumbered bearish stocks.
- OFSS had one of the highest average closing prices.
- Interactive filters allow analysis by company and date.
